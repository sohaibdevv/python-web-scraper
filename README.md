# Python Web Scraper

A robust data acquisition tool built in Google Colab to extract and process web data efficiently.

## Features
- **Resilient Scraping:** Includes headers to mimic browser behavior and avoid blocks.
- **Data Cleaning:** Uses Pandas to structure raw HTML into clean DataFrames.
- **Error Handling:** Built-in try-except blocks to manage network errors and missing data.
- **Export Ready:** Automatically generates a CSV file for downstream analysis.

## Tech Stack
- Python
- BeautifulSoup4 (Parsing)
- Requests (HTTP)
- Pandas (Data Manipulation)
